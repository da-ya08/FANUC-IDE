# FANUC LS Program Editor

![Screenshot](resources/screenshot.png)

Кроссплатформенная IDE для программирования роботов FANUC (файлы LS) с подсветкой синтаксиса.

## Features

- Подсветка синтаксиса для программ FANUC .LS
- Нумерация строк
- Темная/Светлая темы
- Поддержка нескольких языков (английский/русский)
- При открытии скрывает: заголовочную информацию, нумерацию в тексте, точки с запятой, информацию точек.
- Возможность перетаскивания файла .LS на упакованный скрипт в exe, для его открытия
- Загрузка и отправка файлов по FTP. Так же и с реальным роботом

## Installation

1. Клонировать репозиторий:
```bash
git clone https://github.com/da-ya08/FANUC-IDE.git
cd fanuc-ide
```

2. Установить зависимости:
```bash
pip install -r requirements.txt
```

3. Запустить:
```bash
python src/main.py
```

## Get .exe

Так же, Вы можете [скачать .exe](https://github.com/da-ya08/FANUC-IDE/releases/tag/Release).

## Usage

Откройте файл .LS, отредактируйте и сохраните.

## Use FTP

- В меню FTP, функция скачаивания запрашивает ссылку FTP (пример: ftp://admin@127.0.0.1/md%3A%5Cmain.ls)
- Если отсутствует логин в ссылке, он берётся из [conf.py](conf.py), переменной _FTP_DATA_
- Если файл был скачан, то при отправке, отправится в то же место, где он был скачан
- При необходимости, файл можно отправить по иной ссылке "Отправить по ссылке..."
