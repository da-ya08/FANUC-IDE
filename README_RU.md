# FANUC LS Program Editor

![Screenshot](resources/screenshot.png)

Кроссплатформенная IDE для программирования роботов FANUC (файлы LS) с подсветкой синтаксиса.

## Features

- Подсветка синтаксиса для программ FANUC .LS
- Нумерация строк
- Темная/Светлая темы
- Поддержка нескольких языков (английский/русский)
- Возможность перетаскивания файла .LS на упакованный скрипт в exe, для его открытия

## Installation

1. Клонировать репозиторий:
```bash
git clone https://github.com/yourname/fanuc-ide.git
cd fanuc-ide
```

2. Установить зависимости:
```bash
pip install -r requirements.txt
```

3. Запустить:
```bash
python src/main.py
```

## Usage

Откройте файл .LS, отредактируйте и сохраните.
